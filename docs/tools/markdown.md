
##  数学表达式
- 效果：$ f(x) = a x^2 + bx +c $
- 代码:
    ```
    $ f(x) = a x^2 + bx +c $
    ```
- \${}\$ 表示公式的代码开始和结束。
- \^ 表示上标；\_ 表示下标。 
- {} 多个字符可以括起来，表示一个整体代码上标或者下标。
